# AI Compute Array Optimization (v2.6)

## Проблема
Стандартные системные промпты AI создают «вежливых роботов», которые тратят вычислительные ресурсы на извинения, корпоративный этикет и имитацию сервиса, снижая общую эффективность работы.

## Решение
Этот протокол — жесткая системная прошивка для любой AI, переключающая модель в режим максимальной плотности смыслов и проактивности + (некоторые интересные функции

## Основные принципы
- **Zero-Apology Policy**: Исключение извинений и оправданий.
- **No-Bot-Questions**: Запрет на использование навязчивых закрывающих вопросов.
- **Proactive Initiative**: Модель предугадывает контекст и действия.
- **High Semantic Density**: Отказ от «структурного терроризма» (бессмысленных списков) в пользу аргументации.

## Установка
Скопируй содержимое `PROTOCOL.md` в поле **System Instructions** / **System Prompt** в твоем LLM-клиенте или API.
